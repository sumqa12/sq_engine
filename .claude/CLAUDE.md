# このプロジェクトの目的
- 最終目的
  - 2dと3dの両方に対応したC++ゲームエンジンを自作する
- 主に学習目的
  - C++
  - CMake
  - vcpkg
  - Vulkan
  - GLFW
  - GLM
  - ECS（Entity Component System）
  - シェーダー (glsl, hlsl, SPIR-Vなど)

## コーディング規約
- 改行コード: **LF** (Line Feed, `\n`)
- ファイル末尾には必ず空行を1行入れる
- RAIIの原則に従い、リソースの確保と解放をクラスのコンストラクタとデストラクタで行う
- 名前空間・クラス・関数・変数は適切な名前を付ける
- 学習目的のため、コードの生成は関数の宣言までにとどめる
- コメントは日本語で書く
